### ExtraConf Tag
<br />

#### Description
<br />

Specify the location of a file containing a JSON-formatted DataTables configuration to merge with the generated one.

<br />
#### Usage

    <datatables:table>
       ...
       <datatables:extraConf src="..." />
       ...
    </datatables:table>
